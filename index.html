❓ FAQ

Pertanyaan Umum

Apa itu gas fee crypto?
Biaya untuk memproses transaksi di blockchain. Dibutuhkan agar transaksi bisa dijalankan validator.
Minimal beli gas fee?
Rp10.000, maksimal Rp100.000 per transaksi.
Cara top up?
Ketik /topup, masukkan nominal, hubungi admin. Saldo otomatis bertambah.
Blockchain apa saja?
BNB, ETH Base, ETH Arbitrum, Polygon, GRAM, Ronin, Solana.
Jam operasional?
Bot 24 jam. Top up saldo 07:00-22:00 WIB.
Bukti transaksi?
Bot kasih tx hash + link block explorer. Cek juga via /riwayat.
Program referral?
Ketik /referral. Dapat point dari teman (1 point = Rp1). Tukar minimal 100 point.
Transaksi gagal?
Saldo dikembalikan otomatis. Hubungi admin jika masalah berlanjut.
